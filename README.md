# 크로스 플랫폼 앱개발

```
이전에 리액트 네이티브로 제작한 앱입니다.
최종 소스는 아님
```

##  개요

1. 키즈 쇼핑몰을 크롤링하여 자체 DB저장
2. 인기도, 방문수, 찜수 등으로 인기도선정  
   추가로 네이버데이터랩의 랭킹 정보 반영

![demo](/imgs/demo.gif)
